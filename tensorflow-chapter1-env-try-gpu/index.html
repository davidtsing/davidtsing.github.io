深度学习之TensorFlow（1）环境搭建

发表于 2018-04-17 | 分类于 Machine Learning

在Windows平台下安装最新的GPU版本TensorFlow。

安装Anaconda

在以下网站可以下载对应的.exe，安装即可。
https://www.anaconda.com/download/
注意下载对应的版本，比如Anaconda 3.6.4也就对应Python 3.6.4，这时候安装TensorFlow时也必须安装TensorFlow 3.6*.

更新pip

Anaconda安装成功后，pip也就自动安装了。可以更新一下，不然运行命令可能会有警告。

安装GPU最新版TensorFlow

先卸载我已经安装了的CPU版本TensorFlow，

对于GPU版本TensorFlow直接使用命令，

报错了，
notebook 5.4.0 requires ipykernel, which is not installed.
jupyter 1.0.0 requires ipykernel, which is not installed.
jupyter-console 5.2.0 requires ipykernel, which is not installed.
ipywidgets 7.1.1 requires ipykernel>=4.5.1, which is not installed.
缺少，安装ipykernel

安装GPU版本TensorFlow成功之后，尝试跑一下，又报错了
ImportError: Could not find ‘nvcuda.dll’. TensorFlow requires that this DLL be installed in a directory that is named in your %PATH% environment variable. Typically it is installed in ‘C:\Windows\System32’. If it is not present, ensure that you have a CUDA-capable GPU with the correct driver installed.
这个时候虽然已经安装好了TensorFlow，但是因为需要GPU加速，我们还需要安装CUDA和cuDNN(专门为Deep Learning准备的加速库)。

安装CUDA

以下为Windows7 CUDA8.0的安装地址，
https://developer.nvidia.com/compute/cuda/8.0/Prod2/local_installers/cuda_8.0.61_windows-exe
CUDA分为网络版和本地版，网络版是安装时下载，本地版是全部下载完再安装（比较大1G+）。一般使用CUDA8.0，尝试新版本9.0可能会出现意想不到的问题。

变量 配置
CUDA_PATH_V8_0 C:\Program Files\NVIDIA GPU Computing Toolkit\CUDA\v8.0
Path %CUDA_PATH_V8_0%;%CUDA_PATH_V8_0%\bin;%CUDA_PATH_V8_0%\lib\x64

安装cuDNN

https://developer.nvidia.com/cudnn
这个需要在网站注册账号，并填写一个问卷才可以下载。这里下载了cudnn-8.0-windows7-x64-v5.1.zip。
C:\Program Files\NVIDIA GPU Computing Toolkit\CUDA\v8.0
解压zip将得到bin，include和lib三个文件夹，将对应的内容拷贝至以上路径相对应目录下。
我原来下载了cudnn-8.0-windows7-x64-v6.0.zip报错了，
ImportError: Could not find ‘nvcuda.dll’. TensorFlow requires that this DLL be installed in a directory that is named in your %PATH% environment variable. Typically it is installed in ‘C:\Windows\System32’. If it is not present, ensure that you have a CUDA-capable GPU with the correct driver installed.
