Hadoop的前世今生

发表于 2018-04-18 | 分类于 Distributed Computing

要说起Hadoop，我们必然要提到Google。面对大数据，Google采用了分布式存储和并行计算方法，构建起了GFS文件系统、MapReduce计算模型和BigTable非关系型数据库组成的基础平台，用来支持自家的产品服务。

Google公司的三篇文章

虽然Google公司没有公开这三个系统的源码，但是通过这三篇文章，奠定一个领域的基础。

  • The Google File System
  • MapReduce: Simplified Data Processing on Large Clusters
  • Bigtable: A Distributed Storage System for Structured Data

后来，一个叫Doug Cutting的人，根据论文搭建了最初的Hadoop系统。
