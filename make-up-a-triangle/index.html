一个线段上任意取两点，能组成三角形的概率？

发表于 2018-04-16 | 分类于 Mathematics and Algorithm

大前提 三角形任意两边之和大于第三边，即可构成一个三角形。

方法一

假设三角形三边分别为$x$,$y$,$z$，线段的长度为$L$。立即推，$x+y+z = L$，$x+y > z$，$x+z > y$，$y+z > x$，且边长都$>0$
由$x$,$y$,$z$确定一个三维空间，

$x+y+z = L$即为中间大$△ABC$，其与$x+y > z$，$x+z > y$，$y+z > x$，且边长都$>0$几个条件最终确定$△IJK$
根据上图进行概率计算，很显然，概率为$\frac{S_{△IJK}}{S_{△ABC}} = \frac{1}{4}$

方法二

假设三角形三边分别为$x$,$y$,$z$，线段的长度为$L$。立即推，$x+y+z = L$，同时有不等式$x+y < L$，故坐标系中$[x,y]$所构成的总面积并不为$L^2$。而是，$x > 0$，$y > 0$以及$x+y < L$所围成的三角形面积。

由，三角形任意两边之和大于第三边可知，
$x+y > z = L-x-y$，可以化为$y > -x + \frac{L}{2}$
$x+z = x+L-x-y > y$，可以化为$y < \frac{L}{2}$
$y+z = y+L-x-y > x$，可以化为$x < \frac{L}{2}$

根据上图进行概率计算，很显然，阴影部分概率为$\frac{\frac{1}{8}}{\frac{1}{2}} = \frac{1}{4}$
